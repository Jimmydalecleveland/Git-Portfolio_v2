Cottages of Hope

Cottages of Hope is a non-profit organization that helps individuals in need of work or struggling with debt. They wanted WSU to help them create a user registration system that would help them track each individuals progress. Our team volunteered to take on the project for a class, and we selected ASP.NET MVC as our framework. This is my current major project and is still being worked on. I'll continue to post progress updates here.

FancyGiraffe

A Java Enterprise capstone course project to create an inventory management system for Weber county's voting assets. Our team of four utilized JSP pages, servlets, and models with an MVC pattern and Agile workflow to complete the project. We met with our client each week over the course of two months to present the current deliverable and discuss the progress of the project and to make sure he was constantly satisfied with it.

Anything For a Friend

Anything For A Friend is a community fundraising organization started by Becky Anderson. They help tragedy stricken families raise money to help with the financial strain of medical bills amassed while trying to save a loved one's life.

This project was for a capstone course with a large team (over 30 students). We were very excited to be able to help such a great organization
